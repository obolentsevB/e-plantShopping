# Paradise Nursery Shopping Application

A React-based web application for browsing and purchasing houseplants. 

## Features
- **Landing Page:** Elegant introduction with a "Get Started" call to action.
- **Product Listing:** Organized categories of houseplants with dynamic "Add to Cart" functionality.
- **Shopping Cart:** Full CRUD capabilities (Create, Read, Update, Delete) for your plant selections with real-time price calculations.
- **State Management:** Powered by Redux Toolkit.
